Q.2.What isJavaScript Output method?

JavaScript provides several methods to output data to the user

console.log():

>>Used for printing messages to the web console, which is mainly used for debugging.

>>Example: console.log('Hello, world!');

alert():

Displays a pop-up alert box with a specified message and an OK button.

>>Example: alert('Hello');

document.write():

Writes directly to the HTML document. This method is generally discouraged in modern development because it can overwrite the entire document if used after the page has finished loading.

>>Example: document.write('Hello, world!');

>>Example: console.error('This is an error message');
