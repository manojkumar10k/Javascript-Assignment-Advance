Q.3. How to used JavaScript Output method?

console.log()

Purpose: Debugging and logging information to the browser console.

>>console.log('This is a message to the console.');

>>console.log('Variable value:', variableName);

2. alert()

Purpose: Display an alert box with a message to the user. It pauses the script execution until the user acknowledges the alert.

>>alert('This is an alert message!');

3. document.write()

Purpose: Write directly to the HTML document. It is mostly used for simple demonstrations and should be avoided in production code because it can overwrite existing content.

>>document.write('This text will appear on the page.');

4. innerHTML

Purpose: Dynamically change the content of an HTML element. This is a common method for updating content on a webpage.

>>document.getElementById('elementId').innerHTML = 'New content here';

5. console.error() and console.warn()

Purpose: Log error or warning messages to the console. Useful for distinguishing error/warning messages from regular logs.

>>console.error('This is an error message.');

>>console.warn('This is a warning message.');
